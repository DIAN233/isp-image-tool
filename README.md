# ISP 画质分析工具

**自动化画质量化分析工具**，集成「清晰度计算、噪声量化、对比图生成、数据可视化」

## 功能介绍
✅ 批量计算图片清晰度（基于 Canny 边缘检测，数值越高细节越丰富）  
✅ 批量量化图像噪声（基于灰度标准差，数值越低画面越干净）  
✅ 自动生成原图-边缘图对比图，汇报/调试直接使用  
✅ 生成带数值标注的清晰度-噪声对比曲线，直观展示参数优劣  

## 适用场景
- ISP 锐化、降噪参数调试（平衡清晰度与噪声）
- 画质客观评估、批量对比不同算法/参数效果
- 数据可视化展示
